ScheduleAdapter

class ScheduleAdapter(val data: MutableList<ExtendedLesson>? = null) : RecyclerDifferViewListAdapter<ScheduleAdapter.ViewHolder, ExtendedLesson>

Адаптер для отображения расписания занятий.

Наследуется от RecyclerDifferViewListAdapter и реализует его абстрактные методы.

Внутренний класс ViewHolder содержит View-элементы для отображения данных.

Адаптер содержит приватные методы getLessonHoursString, getMarksString, getMarksColor для обработки значений DaybookLessonNoPayment и списка List из MarkV2.
