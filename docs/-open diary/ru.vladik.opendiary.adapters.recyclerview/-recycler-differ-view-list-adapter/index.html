RecyclerDifferViewListAdapter

abstract class RecyclerDifferViewListAdapter<VH : RecyclerView.ViewHolder, T : Any>(dataList: MutableList<T>? = null) : RecyclerView.Adapter<VH>

Базовый класс адаптера для RecyclerView, который использует AsyncListDiffer для сравнения списков элементов. Реализация методов добавления и замены элементов списка может быть переопределена в классе-наследнике.
